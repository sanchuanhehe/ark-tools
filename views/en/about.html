This is the extension for ArkTs Language. Fast way to create, init, build and run ArkTs project.

Requirements

After VsCode 1.65.0

Extension Settings

This extension contributes the following settings:

  • arktsTools.ohpmPath: The absolute path to ohpm
  • arktsTools.hvigorPath: The absolute path to hvigor
  • arktsTools.hdcPath: The absolute path to hdc
  • arktsTools.jdkPath: The absolute path to jdk
  • arktsTools.codelinterPath: The absolute path to codelinter
  • arktsTools.codelinterReloadChange: Refresh struct after file/resources/folder change
  • arktsTools.codelinterCheckAfterSave: Run codelinter after save text document
  • arktsTools.intellisensesModulesPath: The absolute path to intellisenses modules roles. \n WARNING! \n The Location MUST be a FOLDER and file name MUST like "ets_role_***.json"

About Init Build Tool!

Intellisenses Modules Roles

Introduce Role Field

1 - root: Package Name or Modules Name, like 'std' in native module;

2 - digit: check trigger digit. for exmaple, if the value is 1, it'll get the name when you key in "import std.".when value is 2 will happend key in "import std.collection.";

3 - libs: all Modules Name in this Package;
